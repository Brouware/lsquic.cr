# lsquic

Crystal bindings to the excellent [LSQUIC](https://github.com/litespeedtech/lsquic) library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     lsquic:
       github: omarroth/lsquic.cr
   ```

2. Run `shards install`
